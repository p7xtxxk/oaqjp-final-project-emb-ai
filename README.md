# Emotion Detector

## Project Overview
Emotion Detector is an AI-powered web application that analyzes user-provided text and identifies the emotions expressed within it. The application uses the IBM Watson NLP Emotion Prediction model to detect emotions such as anger, disgust, fear, joy, and sadness.

## Features
- Detects emotions from textual input.
- Returns confidence scores for multiple emotions.
- Identifies the dominant emotion.
- Web interface built using Flask.
- Error handling for invalid or blank input.
- Unit tested for reliability.
- Static code analysis compliant.
